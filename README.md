# 🏦 Java Console Bank

> A simple, interactive command-line banking system built for my Week 2 Computer Science coursework at [RWTH Aachen University](https://www.rwth-aachen.de/).

## 📖 Overview
This project is a text-based Java application that simulates a basic ATM or bank terminal. It was created early in my studies to practice fundamental programming concepts such as control flow, switch expressions, and basic error handling. 

Notably, this project utilizes modern **Java 25** features, specifically the `java.io.IO` class for streamlined user input/output, avoiding the need for traditional `Scanner` boilerplate.

## ✨ Features
- **Check Balance:** View current available funds.
- **Deposit Money:** Add funds with an automated 5% transaction fee deduction.
- **Withdraw Money:** Take out funds safely with built-in overdraft protection.
- **Calculate Compound Interest:** Project future balance based on a 4% annual interest rate minus a €2 yearly maintenance fee.

## 🚀 Getting Started

### Prerequisites
Because this project uses the latest I/O features, you will need:
- **Java Development Kit (JDK) 25** or higher.
